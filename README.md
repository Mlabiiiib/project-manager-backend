# Project Manager
This application help user to manage projects.

## Features
- View all projects by their status
- View project by id
- Create project
- Update project
- Delete project

## Used technologies
- Git
- JDK 17
- Maven 3
- Spring Boot
- Spring: Web, JPA
- Lombok
- Docker

## Run project locally
In order to run the project in your local machine:
- Install JDK 17
- Install Maven 3
- Install Docker Desktop
- Run docker/docker-compose.yml: docker-compose up
- Run project: mvn spring:run
